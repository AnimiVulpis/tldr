# figlet

> Display large characters made up of ordinary screen characters.

- Use FIGlet with its default settings:

`figlet {{word_or_sentence}}`

- Use FIGlet with a different font:

`figlet -f {{rev}}`

- Center the output:

`figlet -c {{word_or_sentence}}`

- Use a file as input:

`figlet -p < {{path/to/file}}`
